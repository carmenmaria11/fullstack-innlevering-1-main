# fullstack-innlevering-1
Skole oppgave

## **Plan.md – Kort beskrivelse**
Jeg lager et nettsidesystem for restauranten **Tangen Torv**. Nettsiden skal inneholde info om restauranten, vise menyen, åpningstidene og hvordan man kan ta kontakt. Brukere skal også kunne sende inn en bordbestilling, og ansatte får en egen visning der de kan se og følge opp bestillingene.

## **Teknologier**
* Database: MariaDB (kjøres med Docker og Docker Compose)
* HTML, CSS og JavaScript til frontend
* Node.js + Express til backend (modul 3)
* GitHub brukes til versjonskontroll
* Dokumentasjon skrives i README.md (+ ev. DATABASE.md)

## **Modul 1 – Database**
I denne modulen skal jeg:

* Lage et ER-diagram som beskriver datamodellen
* Lage en SQL-fil som oppretter tabeller og relasjoner
* Lage en egen SQL-fil med eksempler på spørringer (sette inn bestilling, hente data osv.)
* Konfigurere databaseoppsett i Docker

## **Modul 2 – Frontend**
Jeg lager statiske sider som viser planlagt funksjonalitet, med midlertidige/hardkodede data:

* Hjemmeside
* Menyside
* Åpningstider
* Kontaktside
* Bestillingsside (uten databasekobling ennå)
* Ansattside med dummy-informasjon

## **Modul 3 – Backend**
Backenden skal:

* Hente meny og bestillinger fra databasen
* Motta og lagre nye bestillinger
* Gi ansatte oversikt over bestillinger via API-endepunkt
